Use Salad in your command line with XMRig CPU

Made by Angaros#1263 - Last update 09/04/2021 @22:42 UTC+2

Running CLI allows you to:

  • Choose custom CPU usage
  • Set max temperatures and start temperatures
  • Potentially earn a bit more

  • Internet Connection
  • A CPU and 5GB+ RAM
  • 500MB disk space
  • Have installed Salad and mined once (that's how you get the logs, so the Rig and wallet ID)

Start by installing the latest XMRig version, found here. This will install the base XMRig miner with the dependencies and required files. It is a rather small package so it shouldn't take too long. Extract it to your folder of choice.

Here is a guide to finding your Salad logs. You'll need these to find the required IDs.

It should look something like this:

You'll want to keep these for when we do the linking

Go in the XMRig folder you installed in Step 1. You'll find a file named start.bat. You'll want to open it with notepad, or a code editor.

Should it not exist, you can also just create it.

We will need to edit the start file, by adding the correct information:

@echo off
xmrig.exe --donate-level=1 -o <pool>:<pool_port> -u <walletAddress>.<rigId> -k -a randomx --nicehash
pause

First, the pool and port: You'll want to use one of these two:

  • stratum+tcp://randomxmonero.usa.nicehash.com:3385 (US pool)
  • stratum+tcp://randomxmonero.eu.nicehash.com:3385 (Europe pool)

Second, the Wallet & Rig IDs: This is pretty straightforward. Add them to the file like shown in the example. You'll need to use the nicehash values for wallet address and rig ID.

Lastly, since we are uing NiceHash pools, add --nicehash at the end of that string

Here's an example of a completed file:

@echo off
xmrig.exe --donate-level=1 -o stratum+tcp://randomxmonero.eu.nicehash.com:3380 -u 373Ee6DSDgAYNAVSnfgUV99248L6Ab7aUH.p36pcuzwyhvd2fx -k -a randomx --nicehash
pause

Here's some pretty useful XMRig commands you can use:

  • --threads=N : limits CPU usage using threads (if you have 8 threads, setting N to 4 will mean 50% usage)
  • --print-time : if you want to print (show) hashrate at a faster rate, "n" being in seconds
  • --keepalive : will send packets to pool to prevent timeout
  • --background : will run XMRig in the background. Note: Will not appear in the System Tray, you'll have to close it via Task Manager

There are more to be found here for aditional customization.

Save the start.bat file, close it, and run it. It should work if everything was set up properly.

Wait about 15 minutes (can be a bit more, a bit less) and verify that you are getting accepted shares, and then earning on Salad. If not, something went wrong. Verify you follwed all steps and retry once the issue is fixed.

And there you have it! This should be a quick guide to CLI crypto on your Salad account!
