T-REX guide

Made by Angaros#1263 - Last update 15/09/2022 @18:01 UTC+1

Running CLI allows you to:

  • Customise GPU usage
  • Support for older GPUs
  • Set max temperatures and start temperatures
  • Potentially earn a bit more

  • Internet Connection
  • A GPU with 4GB+ VRAM
  • 500MB disk space
  • Have installed Salad (0.5.6 if you wish to use Nicehash or Ethermine pools)
  • To have mined once before (This is to generate your Rig and wallet ID, shown in Salad's logs in later steps)
  • If using the TREX LHR unlock, you'll need to have the driver v512.77.

Start by installing the latest T-REX version, found here. This will install T-REX miner. It is a rather small package, so it shouldn't take too long. Extract it to your folder of choice.

You would also want to whitelist the folder that you place T-REX in, just to ensure that your antivirus doesn't decide to interfere.
Here are some guides on how to whitelist a folder with most antivirus.

Here is a guide to finding your Salad logs. You'll need these to find the required IDs.

Note that Salad 1.0 no longer gives the required IDs to mine with Nicehash and Ethermine in the logs - If you wish to use either of these pools, you'll have to use Salad 0.5.6 to get the IDs required for these pools.

You may be able to still get your Prohashing IDs in Salad 1.0 by mining at least once with it, heading to your Salad logs folder, opening the folder for one of the miners (e.g. T-Rex), opening the latest log inside said folder, then at the very top you should see something like "-u salad -p (A bunch of random characters)".

It should look something like this:

You'll want to keep these for when we do the linking

- Go in the T-REX folder you installed in Step 1. You'll find various files here, most commonly those ending with '.bat' (Batch Files). You can choose to either edit one of these, or just create a fresh new batch file.

- You can optionally delete any .bat files that you are not using, to clear some clutter.

- If you choose to create a new .bat file, create any text file (right click --> "New" --> "Text Document"), then save it as start.bat

- At the end of the day, you're going to end up with a .bat file that should look something like this:

Nicehash:

t-rex.exe --algo <algorithm> --url <pool> --user <Wallet Address>.<Rig ID>
pause

Ethermine:

t-rex.exe --algo <algorithm> --url <pool> --user <Wallet Address> --worker <Worker ID>
pause

Prohashing:

trex.exe --algo <algorithm> --url <pool> --user salad -pass <Your Prohashing ID>
pause

- First, the algorithm. There are 2 options:

  • Ravencoin (4GB+ GPUs): kawpow
  • Ethereum Classic (4GB+ GPUs): etchash

- Second, the pool: You'll want to use one of the following:

  • stratum+ssl://kawpow.auto.nicehash.com:443 - For Nicehash KawPow. If you have issues with SSL, you can alternatively use stratum+tcp://kawpow.auto.nicehash.com:9200
  • eu1-etc.ethermine.org:4444 For Ethermine ETC (Ethereum Classic). You can optionally change locations to the one closest to you: eu1, us1, asia1.
  • stratum+tcp://prohashing.com:3361 For Prohashing KawPow (Ravencoin).

- Lastly, the Wallet & Rig IDs: This is pretty straightforward. Add them to the file like shown in the example. You'll need to use the Nicehash values for the Nicehash format, the Ethermine values for the Ethermine format, and the Prohashing values for the Prohashing format.

Ensure that the end of your 'really long text' Prohashing ID has the correct algorithm! For KawPow it should be ,a=kawpow.
Though if this doesn't work, try removing the ',a' algorithm indicator. It should work without it.


- Here's an example of a completed KawPow (Ravencoin) file for Nicehash:

t-rex.exe --algo kawpow --url stratum+ssl://kawpow.auto.nicehash.com:443 --user 33kJvAUL3Na2ifFDGmUPsZLTyDUBGZLhAi.8sqe3di5hfeeonq
pause

- And here's an example of a completed ETC (Ethereum Classic) file for Ethermine:

t-rex.exe --algo etchash --url eu1-etc.ethermine.org:4444 --user 6ff85749ffac2d3a36efa2bc916305433fa93731 --worker 8sqe3di5hfeeonq
pause

Here's some pretty useful T-REX commands you can use:

  • --intensity n : Change usage value "n", from 8 (lowest) to 25 (highest). Default is auto (Recommended)
  • --quiet : Show no GPU stats at all in the CMD window.
  • --devices : A comma separated list of which GPUs to use. Device IDs start counting from 0.
  • --temperature-limit x : Stops the miner when that temperature (x) is reached.
  • --temperature-start y : Starts the miner back up once GPU temperature has cooled down to (y).

The full in-depth list of T-REX miner arguments can be found Here.

- Here's an example of a completed KawPow (Ravencoin) file for Nicehash with extra commands:

t-rex.exe --algo kawpow --url stratum+ssl://kawpow.auto.nicehash.com:443 --user 33kJvAUL3Na2ifFDGmUPsZLTyDUBGZLhAi.8sqe3di5hfeeonq --devices 0,2,3 --intensity 18
pause

Now just save that .bat file, close it, and run it. It should work if everything was set up properly.

Wait about 15 minutes (can be a bit more, a bit less) and verify that you are getting accepted shares, and then earning on Salad. If not, something went wrong. Verify you followed all steps and retry once the issue is fixed.
