Go in the PhoenixMiner folder you installed in Step 1. You'll want to make a new file named start.bat (or any name you want, I just chose start for conveniency). To do that, create any text file (right click --> "New" --> "Text Document"), then save it as start.bat

And next you'll want to start editing it.

On the first line, type: PhoenixMiner.exe -pool ssl://us1.ethermine.org:5555 -pool2 ssl://eu1.ethermine.org:5555 -wal {ETH wallet and rig}

Remember to replace the ETH address with your own ETH wallet address in the -wal option, or you won't be getting earnings

And on the second line, just add pause

The final result of the bat file should look like:

Should you be closer to Europe, your primary pool should be ssl://eu1.ethermine.org:5555, and if you're closer to North America, then use ssl://us1.ethermine.org:5555

Because of the way Phoenix is built, there are tons of customization options you can add to manage mining. Here are a few of the most useful commands:

  • -gpow <percent> where percent is the percentage you want your GPU to mine at (if you want to use only 50% of your GPU, type -gpow 50)
  • -powlim <n> where n is the power limit you want (between -75 to 75)
  • -cclock <MHz> for AMD, enter a core clock in MHz, for NVIDIA, use relative values (-300 / +400)
  • -mclock <MHz> sets GPU memory in MHz
  • -fanmin <percent> where percent is the percentage you want your fans to run at MINIMUM (if you want them to always run somewhere above 70%, type -fanmin 70)
  • -fanmax <percent> same as above, but determines a limit to max fan speeds.
  • -tstop <degrees> where degrees is, in celsius, the temperature at which your GPU will stop if it reaches it. It will stop once it reaches that temperature (if you don't want your GPU to go above 70 °C, type -tstop 70)
  • -tstart <degrees> where degress is, in celsius, the temperature at which your GPU will start if it reaches it. It will start once it falls to that temperature (if you want to start your GPU when it reaches 50 °C, type -tstart 50)

You'll usually want to use tstop and tstart together, as once started it won't stop, and once started it won't stop unless started. Aka, if it reaches tstop limit, it will stop for always unless there is a tstart to get it working again at some point.

General info for the additional commands:

  • You'll want tstop to be higher than tstart.
  • You'll want fanmin to be smaller than fanmax.
  • You'll want possible temperatures for tstart and tstop. (say, if your GPU is unable to go below 50°C, don't set a tstart at 30°C, otherwise it will never happen, same for tstop.)
