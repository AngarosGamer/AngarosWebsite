T-REX guide

Made by Angaros#1263 - Last update 09/04/2021 @22:42 UTC+2

Running CLI allows you to:

  • Custom GPU usage
  • Support for older GPUs
  • Set max temperatures and start temperatures
  • Potentially earn a bit more

  • Internet Connection
  • A GPU with 3GB+ VRAM
  • 500MB disk space
  • Have installed Salad 0.5.6 - 0.5.8, found here. Also to of mined once (that's how you get the logs, so the Rig and wallet ID)

Start by installing the latest T-REX version, found here. This will install T-REX miner. It is a rather small package so it shouldn't take too long. Extract it to your folder of choice.

Here is a guide to finding your Salad logs. You'll need these to find the required IDs.

Salad version 0.5.6 - 0.5.8 will be required to do this. You can find theses here, Salad github.

It should look something like this:

You'll want to keep these for when we do the linking

Go in the T-REX folder you installed in Step 1. You'll find multiple files named various coins. You'll want to open one of them, of your choice, with notepad, or a code editor.

Should it not exist, you can also just create it.

We will need to edit that file, by adding the correct information:

t-rex.exe --algo <algorithm> --url <pool>:<port> --user <walletAddress>.<rigId>
pause

First, the algorithm. There are 3 options:

  • Ethereum (6GB+ GPUs): ethash
  • KawPow (4GB+ GPUs): kawpow
  • Ethereum Classic (3GB+ GPUs): etchash

Second, the pool: You'll want to use one of these three:

  • stratum+tcp://daggerhashimoto.eu.nicehash.com:3353 (EU pool, you can replace eu with usa, eu-west, eu-north, usa-west, usa-east)
  • stratum+tcp://kawpow.eu.nicehash.com:3385 (EU pool, you can replace eu with usa, eu-west, eu-north, usa-west, usa-east)
  • ssl://eu1-etc.ethermine.org:5555 (EU pool, you can replace eu1 with us1 or asia1)

Lastly, the Wallet & Rig IDs: This is pretty straightforward. Add them to the file like shown in the example. You'll need to use the nicehash values for the KawPow pool, the ethermine one for ETH and ETC.

Here's an example of a completed ETH (Ethereum) file:

t-rex.exe --algo ethash --url stratum+tcp://daggerhashimoto.eu.nicehash.com:3353 --user 33kJvAUL3Na2ifFDGmUPsZLTyDUBGZLhAi.8sqe3di5hfeeonq
pause

Here's some pretty useful T-REX commands you can use:

  • --intensity : Change usage value, from 8 (lowest) to 25 (highest)
  • --quiet : No GPU stats at all.
  • --time-limit : Miner shutdown interval in seconds.
  • temperature-limit : Stops the miner when that temperature is reached.
  • --temperature-start : GPU temperature to enable card after disable.

There are more to be found here for additional customization.

Save the bat file you chose, close it, and run it. It should work if everything was set up properly.

Wait about 15 minutes (can be a bit more, a bit less) and verify that you are getting accepted shares, and then earning on Salad. If not, something went wrong. Verify you follwed all steps and retry once the issue is fixed.
