GMiner guide

Made by Angaros#1263 - Last update 09/04/2021 @22:42 UTC+2

Running CLI allows you to:

  • Custom GPU usage
  • Support for older GPUs
  • Set max temperatures and start temperatures
  • Potentially earn a bit more

There's a video that will explain in detail the process of setting up GMiner. Shows the different file formats you can use for the various coins and algos you can use.

  • Internet Connection
  • A GPU with 2GB+ VRAM
  • 500MB disk space
  • Have installed Salad and mined once (that's how you get the logs, so the Rig and wallet ID)

Start by installing the latest GMiner version, found here. This will install the base GMiner miner with the dependencies and required files. It is a rather small package so it shouldn't take too long. Extract it to your folder of choice.

Here is a guide to finding your Salad logs. You'll need these to find the required IDs.

It should look something like this:

You'll want to keep these for when we do the linking

Go in the GMiner folder you installed in Step 1. You'll find multiple files named various coins. You'll want to open one of them, of your choice, with notepad, or a code editor.

Should it not exist, you can also just create it.

We will need to edit that file, by adding the correct information:

miner.exe -a <algorithm> -s <pool>:<port> -u <walletAddress>.<rigId> -pers auto
pause

First, the algorithm. You'll want to use any of these 5 different:

  • ethash (Ethereum) - 6GB+ cards
  • etchash (Ethereum Classic) - 3GB cards
  • kawpow (Raven / Monero) - 4GB cards
  • beamhashIII (Beam V3) - 3GB cards
  • 144_5 (Bitcoin Gold - 2GB cards

Second, the pool and port: You'll want to use one of these five (they correspond to the algorithms, in order):

  • eu1.ethermine.org:5555 --ssl (EU pool, you can replace eu1 with us1 or asia1)
  • eu1-etc.ethermine.org:5555 --ssl (EU pool, you can replace eu1 with us1 or asia1)
  • kawpow.eu.nicehash.com:3385 (EU pool, you can replace eu with usa, eu-west, eu-north, usa-west, usa-east)
  • beamv3.eu.nicehash.com:3387 (EU pool, you can replace eu with usa, eu-west, eu-north, usa-west, usa-east)
  • zhash.eu.nicehash.com:3369 (EU pool, you can replace eu with usa, eu-west, eu-north, usa-west, usa-east)

Lastly, the Wallet & Rig IDs: This is pretty straightforward. Add them to the file like shown in the example. You'll need to use the nicehash values for nicehash pools, and the Ethermine wallet and rig for ethermine pools.

Here's an example of a completed Ethereum file:

miner.exe -a ethash -s eu1.ethermine.org:5555 --ssl -u 0x6ff85749ffac2d3a36efa2bc916305433fa93731.p36pcuzwyhvd2fx
pause

Here's some pretty useful GMiner commands you can use:

  • --intensity : Change usage value, in percentage. Default is 100%
  • --cclock : Change core clocks - NVIDIA in relative values, AMD in absolute values.
  • --mclock : Change memory clocks - NVIDIA in relative values, AMD in absolute values.

There are more to be found here for aditional customization.

Save the bat file you chose, close it, and run it. It should work if everything was set up properly.

Wait about 15 minutes (can be a bit more, a bit less) and verify that you are getting accepted shares, and then earning on Salad. If not, something went wrong. Verify you follwed all steps and retry once the issue is fixed.
